# Skills for Runwaize tools

Agent skills for the [Runwaize](https://runwaize.com) ecosystem. Each subfolder is a self-contained skill that a coding agent (Claude Code, Cursor, Codex, Gemini CLI, etc.) can load and follow.

## Skills

| Skill                                                   | Purpose                                                                |
| ------------------------------------------------------- | ---------------------------------------------------------------------- |
| [`supervaizer_integration`](./supervaizer_integration/) | Integrate an existing Python agent with the Supervaizer controller     |
| [`supervaize_access`](./supervaize_access/)             | Operate against an existing Supervaize workspace from a chatbot or CLI |

## Documentation

Full documentation: https://docs.runwaize.com/docs/integrations/skills/
