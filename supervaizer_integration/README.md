# Supervaizer Integration

Integrate any existing Python agent with the [Supervaizer](https://github.com/supervaize/supervaizer) controller and the [Supervaize](https://supervaize.com) platform.

## Install

In Claude Code:

```
/plugin marketplace add runwaize/runwaize_skills
/plugin install supervaizer_integration
```

Or manually:

```bash
git clone https://github.com/runwaize/runwaize_skills.git
```

Then point your coding agent to the skill definition in [`SKILL.md`](./SKILL.md).

## Documentation

Full documentation: https://docs.runwaize.com/docs/integrations/skills/supervaizer-integration
