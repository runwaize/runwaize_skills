# Supervaize Access

Operate against an existing [Supervaize](https://supervaize.com) workspace from a chatbot or CLI agent. Register, create missions, start jobs, inspect cases/steps, and check job status.

## Install

In Claude Code:

```
/plugin marketplace add runwaize/runwaize_skills
/plugin install supervaize-access@runwaize-skills
```

Or manually:

```bash
git clone https://github.com/runwaize/runwaize_skills.git
```

Then point your coding agent to the skill definition in [`SKILL.md`](./SKILL.md).
